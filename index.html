Yang Akan Anda Pelajari

Kurikulum yang dirancang khusus untuk mengembangkan skill web development dari nol

Belajar Dasar HTML

Kuasai pondasi pembuatan website dengan mempelajari elemen-elemen dasar HTML untuk menyusun struktur halaman.

  • Pengenalan tag HTML
  • Heading, paragraf, dan teks
  • Link dan gambar

Belajar Dasar CSS

Pelajari cara mempercantik tampilan website dengan styling CSS, mulai dari warna, font, hingga layout dasar.

  • Warna, background, dan font
  • Box model dan spacing (margin & padding)
  • Flexbox untuk layout sederhana

Belajar Dasar JavaScript

Tambahkan interaktivitas pada website dengan mempelajari dasar-dasar JavaScript, dari manipulasi elemen hingga event handling.

  • Pengenalan variabel, tipe data, dan operator
  • Fungsi dan kontrol alur (if, loop)
  • Event handling (click, input, submit)
